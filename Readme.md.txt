# IPL 2026 Dashboard


## Overview:
 
This Power BI dashboard provides an interactive analysis of the Indian Premier League (IPL) using historical match data. It enables users to explore team performance, player statistics, season trends, and match outcomes through dynamic visualizations and filters.


# Objectives:

- Analyze IPL teams performance over season 2026.
- Compare runs and wickets statistics.
- Identify top-performing players.
- Track match outcomes and winning trends.


## Tools :
- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Excel / CSV Dataset


## 📊 Dashboard Features


-Total no.of matches
-Won team 
-Venue wise matches
-Team wise wininng analysis
-Perormance trends: runs v/s wickets
-Player of the match analysis
-Interactive Filters
